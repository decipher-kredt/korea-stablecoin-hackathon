# 대한민국 스테이블코인 해커톤 - Team Decipher

**🇰🇷 대한민국 스테이블코인 해커톤 참여작**

## 📋 프로젝트 개요

이 프로젝트는 스테이블코인의 실제 사용 사례를 보여주는 3개의 핵심 데모를 포함한 종합 플랫폼입니다. Kaia 블록체인 기반의 스마트 컨트랙트를 통해 안전하고 투명한 금융 서비스를 제공합니다.

## 🎯 3가지 핵심 데모

### 1️⃣ 스테이블코인 예금 시스템

**고객과 은행 양면의 디지털 예금 서비스**

- **고객 화면**: 스테이블코인 예치/출금, 실시간 이자 확인
- **은행 화면**: 스테이블코인 발행, 원금+이자 송금 관리
- **핵심 기능**:
  - 연 3.0% 고정 금리 (365일 잠금)
  - 실시간 이자 계산 및 잔액 조회
  - 투명한 블록체인 기반 예금 관리

### 2️⃣ 스테이블코인 영수증 발행 시스템

**스테이블코인 결제 시 소득공제용 스테이블코인 영수증 자동 발행**

- **결제 프로세스**: 스테이블코인으로 상품/서비스 결제
- **영수증 발행**: 국세청 연동 스테이블코인 영수증 자동 생성
- **핵심 기능**:
  - 스테이블코인 결제 내역 자동 추적
  - 소득공제 적용 가능한 공식 영수증
  - 기존 카드 결제와 동일한 세무 처리

### 3️⃣ 이커머스 셀러 정산 시스템

**이커머스 플랫폼의 판매자 자동 정산 서비스**

- **쇼핑몰 시뮬레이션**: 다양한 브랜드 상품 구매 체험
- **스마트 정산**: 구매 완료 시 각 판매자에게 자동 정산
- **핵심 기능**:
  - 멀티 셀러 동시 정산 처리
  - 수수료 자동 계산 및 분배
  - 투명한 정산 내역 추적

## 🏗️ 아키텍처

### Frontend (`web/`)

- **프레임워크**: React 19 + TypeScript
- **스타일링**: Tailwind CSS + Framer Motion
- **웹3 연동**: Ethers.js, Web3.js
- **UI 컴포넌트**: shadcn/ui 기반 커스텀 컴포넌트

### Smart Contracts (`contracts/`)

- **개발 환경**: Foundry
- **주요 컨트랙트**:
  - `Vault.sol`: 예치/출금 핵심 로직 (3.0% 연이율, 365일 잠금)
  - `VaultManager.sol`: 볼트 관리 시스템
  - `PaymentSystem.sol`: 결제 처리 시스템
  - `ECommerce.sol`: 전자상거래 통합
  - `Token.sol`: ERC20 스테이블코인 구현

### 로컬 실행 방법

#### 필수 요구사항

- Node.js 16+
- Foundry
- MetaMask 등 웹3 지갑

#### 설치 및 실행

1. **저장소 복제**

   ```bash
   git clone <repository-url>
   cd kaia-hackathon
   ```

2. **스마트 컨트랙트 설정**

   ```bash
   cd contracts
   forge install
   forge build
   forge test
   ```

3. **Frontend 애플리케이션 실행**

   ```bash
   cd web
   npm install
   npm start
   ```

4. **브라우저에서 접속**
   - http://localhost:3000

#### 데모 이용 가이드

1. **지갑 연결**: MetaMask로 Kaia 테스트넷 연결
2. **예금 시스템**: 고객/은행 화면 전환하며 예금 프로세스 체험
3. **스테이블코인 영수증**: 스테이블코인 결제 후 영수증 발행 확인
4. **이커머스 정산**: 쇼핑 후 자동 셀러 정산 프로세스 확인

## 💡 데모별 주요 기능

### 🏦 예금 시스템

- ✅ **듀얼 인터페이스**: 고객용/은행용 화면 분리
- ✅ **스테이블코인 예치**: 사용자 자금을 안전하게 보관
- ✅ **이자 적립**: 연 3.0% 고정 이율 (365일 잠금)

- ✅ **결제 내역 추적**: 블록체인을 통한 투명한 거래 기록

### 🛒 이커머스 정산 시스템

- ✅ **멀티 브랜드 쇼핑**: 실제 쇼핑몰과 유사한 구매 경험
- ✅ **스마트 정산**: 구매 완료 시 자동 판매자 정산
- ✅ **수수료 관리**: 플랫폼 수수료 자동 계산 및 분배
- ✅ **실시간 정산**: 즉시 정산으로 판매자 현금흐름 개선
